Importer des offres (Excel)

Télécharger le template Excel

Colonnes attendues (feuille 1) :

  • OffreNom, Description, DateDebut (yyyy-MM-dd), DateFin (yyyy-MM-dd), Active (true/false)
  • ProduitCode (ou ProduitNom), Prix, Remise%, Min, Max, Colisage (optionnel)
  • Gamme (optionnel : id ou nom — doit être autorisée pour le DSM)

Plusieurs lignes peuvent appartenir à la même offre, regroupées par (OffreNom + DateDebut + DateFin).
